<h1 align="center">🎵 SonicFlow: Real-time Music Streaming ✨</h1>

<p align="center">
  A high-performance, full-stack music application featuring real-time social integration, built with the MERN stack.
</p>

![SonicFlow Demo](/frontend/public/screenshot-for-readme.png)

## 🌟 Key Features

- 🎸 **Seamless Playback:** Listen to music, skip tracks, and manage your queue.
- 🔈 **Precision Audio:** Interactive volume control with a smooth slider interface.
- 🎧 **Admin Control:** Dedicated dashboard for managing the library (albums and songs).
- 💬 **Real-time Social:** Integrated chat system to connect with other listeners.
- 👨🏼‍💼 **Presence Tracking:** See who's online or offline in real-time.
- 👀 **Live Activity:** Watch what your friends are listening to as it happens.
- 📊 **Analytics:** Data aggregation for insights into your music library.
- 🚀 **Modern Tech:** Built with React, TypeScript, and MongoDB.

## 🛠️ Local Installation & Setup
### 1. Backend Configuration
Create a `.env` file in the `backend` folder and add your credentials:

```bash
PORT=5000
MONGODB_URI=your_mongodb_uri
ADMIN_EMAIL=your_email
NODE_ENV=development

CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret
CLOUDINARY_CLOUD_NAME=your_cloud_name

CLERK_PUBLISHABLE_KEY=your_clerk_pub_key
CLERK_SECRET_KEY=your_clerk_secret_key
